# StudioCDictionary
A Digital Dictionary of BYUtv's Studio C


One of BYUtv's most popular original series, the sketch-comedy show Studio C brings you memorable characters like Scott Sterling, the Bisque Man, and the Awkward Avoidance Viking.
Join the cast--including Whitney Call, Mallory Everton, Jason Gray, and Matt Meese--to enjoy more hilarious sketches and characters.

The Studio C Digital Dictionary is a windows desktop application designed to make it as easy as possible to find any memorable moment, quote or sketch in Studio C's first nine seasons.
Search quotes, full sketch transcripts and characters to find all of Studio C's best content assembled into one all-inclusive dictionary.
